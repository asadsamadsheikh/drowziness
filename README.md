Drowziness — AI Eye Drowsiness Detector
Drowziness is a free, browser-based AI drowsiness detection system designed for two types of users — drivers and students. Drivers can use it 
to stay alert during long trips by receiving real-time audio and visual alerts the moment drowsiness is detected through their webcam. 
Students can use it to monitor their focus during study sessions, track how long they stay attentive, and receive smart break reminders to 
improve productivity.
The application works entirely in the browser using your device's webcam. No video is recorded or uploaded — all processing happens locally 
on your device, making it completely private and safe to use. Users simply sign up with their role, allow camera access, and the AI begins 
monitoring eye movement, blink rate, and head position in real time.
The project is built using HTML, CSS, and JavaScript for the frontend, with a Python backend planned for user authentication and session 
data storage. The interface is fully responsive and works on both desktop and mobile browsers. Future plans include weekly focus reports, 
a parent and teacher portal for students, a Pomodoro-style break timer, and integration with a trained deep learning model for even more 
accurate drowsiness detection.
This project was created with the goal of making road safety and student focus tools accessible to everyone, completely free of charge.
